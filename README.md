# ParkingABC Demo

This repository showcases a demo using `Kotlin Multiplatform Mobile` and `Kotlin Spring Boot`. This demo can also be
used as template for future projects needing `Android and iOS` implementation.

## Goals

- Two Apps
    - User App
    - Parking Owner App
- QR Code Scanning
- Invoicing
- Real Time Map
- Spring Boot Backend
